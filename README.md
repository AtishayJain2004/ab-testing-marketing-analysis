# A/B Testing Analysis for Marketing Campaign Optimization

## Objective
To evaluate whether showing ads improves user conversion rates compared to a control group.

## Dataset
The dataset contains user-level data including test group assignment (ad/psa), conversion outcome, and ad exposure details.

## Methodology
- Exploratory Data Analysis (EDA)
- Hypothesis Testing using Z-test
- Comparison of conversion rates between groups

## Results
The ad group showed a higher conversion rate compared to the control group.  
The p-value was extremely small, indicating that the result is statistically significant.

## Conclusion
Running ads leads to improved conversions and has a positive impact on user behaviour.

## Business Impact
This analysis supports data-driven decision-making for marketing investments and campaign optimization.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
